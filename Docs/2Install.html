Instalação

Para que a FaceStudy seja utilizada ela precisa de bibliotecas auxiliares que precisam ser importadas previamente. A primeira é a asyncio que permitirá a utilização de funções assíncronas que poderão ser úteis dependendo da aplicação. A segunda é a cv2 para que seja possível fazer a análise facial em tempo real com o uso da webcam. E a última é a DeepFace que é a responsável por fazer a análise das informações e devolver os dados desejados.

Setup - Importando as bibliotecas externas necessárias:

import asyncio

import cv2

from deepface import DeepFace

Importando a versão sincrona e assíncrona da FaceStudy

from .FaceStudy import FaceStudy, AsyncFaceStudy
