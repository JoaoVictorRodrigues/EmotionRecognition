Funções

Todas as funções da biblioteca podem ser realizadas de maneira síncrona e assíncrona para que atenda ao interesse do usuário quanto a aplicação.

Essa função é responsável por determinar a característica desejada da foto, idade, gênero, emoção ou raça. Isso é feito mudando o valor do analyze_type para age, gender, emotion ou race para cada um respectivamente.

Análise assíncrona com callback

async def test_async_with_callback():

async_face_study = AsyncFaceStudy(analyze_type='age', window='Async With Callback')

await async_face_study.run()

Análise assíncrona sem callback

async def test_async_without_callback():

async_face_study = AsyncFaceStudy(analyze_type='age', window='Async Without Callback')

await async_face_study.run()

Análise síncrona

sync_face_study = SyncFaceStudy(analyze_type='age', window='Sync Face Study')

sync_face_study.run()

Run
Esse comando é responsável por rodar o programa, uma janela irá abrir e sua webcam será ligada, a partir desse momento a função começará a te devolver os resultados encontrados.
